#  Smart Student Task Manager (Java)

##  Overview

The Smart Student Task Manager is a Java-based console application designed to help students manage their daily tasks, assignments, and deadlines efficiently. It provides a simple interface to add, view, and delete tasks while storing data persistently using file handling.

---

##  Problem Statement

Students often forget deadlines and struggle to organize their tasks effectively. This leads to missed submissions and increased stress.

---

##  Solution

This project provides a lightweight task management system that allows users to:

* Add tasks with deadlines and priorities
* View all tasks in an organized manner
* Delete completed or unnecessary tasks
* Store tasks permanently using file handling

---

##  Features

*  Add new tasks
*  View all tasks
*  Delete tasks
*  Persistent storage using file handling
*  Simple and user-friendly console interface

---

##  Technologies Used

* Java
* Object-Oriented Programming (OOP)

##  How to Run the Project

1. Open terminal in project directory

2. Navigate to src folder:
   cd src

3. Compile the program:
   javac *.java

4. Run the program:
   java Main

---

##  Sample Usage

1. Add Task → Enter title, deadline, priority
2. View Tasks → Displays all tasks
3. Delete Task → Removes selected task
